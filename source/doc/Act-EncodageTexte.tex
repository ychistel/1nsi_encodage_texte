\begin{NSI}
{Activité}{Encodage du texte}
\end{NSI}


Les ordinateurs ne manipulent que des valeurs binaires, à savoir des $0$ et des $1$. Il a donc été nécessaire de traduire les textes en binaire afin d'envoyer des mails, rédiger des documents ou créer des pages web, etc.\medskip

Selon le lieu géographique et la langue parlée, les caractères utilisés doivent être transformés en un codage binaire. Les codages des caractères ont beaucoup évolué avec le temps et ont été regroupés sous forme de tables.

Nous allons nous intéresser à trois tables qui ont marqué l'évolution de l'encodage des caractères. \bigskip


\textbf{Avec un éditeur de texte, vous allez réaliser une présentation des trois tables de caractères en suivant le plan donné ci-dessous.}\medskip

\begin{enumerate}
\item La première table de caractères est ASCII.
\begin{enumerate}
\item Que signifie l'acronyme ASCII ? Quelle est son origine ? Combien de caractères contient-elle ?
\item Insérer dans votre document une image de la table ASCII.
\item Quel est le nombre de bits utilisés pour chaque caractère ? Donner une plage (intervalle) du premier au dernier caractère ASCII.
\item Á quoi correspondent les 32 premiers caractères de cette table ? Et le dernier.
\item Quels caractères correspondent aux codes hexadécimaux 6E, 53 et 49  ? Donner pour chaque caractère sa valeur binaire et décimale.
\item Quel est le principal problème de cette table de caractères pour notre langue ?
\end{enumerate}\medskip

\item Une seconde table introduite par l'ISO.
\begin{enumerate}
\item Que signifie l'acronyme ISO ?
\item Combien de tables contient la norme 8859 ?
\item Combien de caractères peut contenir une table à la norme ISO-8859 ?
\item Quel est le nombre de bits utilisés par chaque caractère ? Donner une plage (intervalle) du premier au dernier caractère ISO-8859 non ASCII.
\item Quelles ont été les deux table ISO-8859 utilisées en France ? Pourquoi ?
\end{enumerate}\medskip

\item La norme Unicode

\begin{enumerate}
\item Quel est l'avantage de la norme Unicode sur la norme ISO ?
\item Qu'est-ce qu'un point de code dans la norme Unicode ? Comment se note-t-il ?
\item Quelles sont les points de code des caractères ASCII ?
\item Quel est le point de code associé au caractère È (alt+212) ? æ (alt+145) ? \pounds ?
\item Quel est le caractère associé au point de code est U+20AC ?
\item UTF-8 encode en binaire les caractères à partir de leur point de code. Combien d'octets sont nécessaires pour l'encodage d'un caractère ?
\item Donner selon les plages de valeurs Unicode l'encodage d'un caractère en UTF-8.
\end{enumerate}
\end{enumerate}
